Data safety

The Google Mobile AdMob SDK collects and shares some data types automatically for advertising, analytics, and fraud prevention purposes. This includes the device’s IP address, user product interactions, diagnostic information, and device and account identifiers. All of the user data collected by Google Mobile Ads SDK is encrypted in transit using the Transport Layer Security (TLS) protocol.
